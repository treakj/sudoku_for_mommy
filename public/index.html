ℹ️ Como Jogar

⌨️ Teclado

  • 1-9 Inserir números
  • Shift + 1-9 Modo notas
  • Q Alternar modo notas
  • C Alternar entre cores
  • Space Aplicar cor selecionada
  • Ctrl+Z Desfazer
  • Ctrl+Y Refazer
  • H Dica
  • Delete Apagar
  • ↑↓←→ Navegar

🖱️ Mouse/Touch

  • Clique para selecionar célula
  • Clique em número para destacar
  • Mobile: Use painel inferior

  • 🔢 Contador de números (canto)
  • 📝 Sistema de notas
  • 🎨 Sistema de cores
